# chatroom-v2
 Chatting platform based on NodeJs and ReactJs with the help of socket.io. 
 
 ## Live Demo
 https://chatroom-my.herokuapp.com
 
## Framework/Library
* React
* express
* socketio
* tailwindcss
* emoji-picker-react
* react-icons
* nodemon
