# chatroom-v2
 Chatting platform based on NodeJs and ReactJs. 
 Connect user based on room name input using Socket.io
 
## Framework/Library
* React
* express
* socketio
* tailwindcss
* emoji-picker-react
* react-icons
* nodemon
